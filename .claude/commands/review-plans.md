# Review Project Plans

List and summarize all implementation plans from both `.claude/plans/` and `openspec/plans/`.

## Instructions

1. List all plans with their last modified date
2. For each plan, extract and show:
   - **Title** (from H1 heading)
   - **Status** (if specified)
   - **Summary** (first paragraph or description)
3. Group by location (Claude plans vs OpenSpec plans)
4. Sort by most recently modified

## Output Format

```
## Recent Plans

### .claude/plans/
| Plan | Modified | Status |
|------|----------|--------|
| [name] | date | status |

### openspec/plans/
| Plan | Modified | Status |
|------|----------|--------|
| [name] | date | status |

## Plan Summaries
[Brief summary of each plan's purpose]
```
