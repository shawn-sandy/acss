# Details Styles

`details.scss` styles native `<details>`/`<summary>` elements with animated disclosure.

## CSS Custom Properties

- Container tokens: `--details-border`, `--details-direction`, `--details-display`, `--details-gap`, `--details-width`, `--details-radius`, `--details-max-height-closed|open`.
- Summary tokens: `--summary-display`, `--summary-justify`, `--summary-align`, `--summary-gap`, `--summary-transitions`, padding aliases, and cursor control.

## Base Styling

- Details render as flex containers with border rails and `interpolate-size: allow-keywords` for smooth height transitions.
- Summary hides default markers, aligns content via flex, and gains hover cursor and `focus-within` underline/background.
- Content sections use shared padding and transition with `@starting-style` support.

## States

- `[open]` lifts `max-height` to `max-content`, adds a top border to summary, and caps section height with `--details-max-height-open`.
- First/last/only items adjust border radius; adjacent details drop double borders.

## Usage

```tsx
<details>
  <summary>More information</summary>
  <section>
    <p>Collapsible copy styled with shared paddings.</p>
  </section>
</details>
```
