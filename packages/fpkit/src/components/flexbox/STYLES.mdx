# Flex Styles

Documented utility classes generated by `flex.scss`. These classes power the `Flex`, `Flex.Item`, and `Flex.Spacer` components and can be used directly in markup.

## CSS Custom Properties

- `--flex-gap-xs|sm|md|lg|xl` fluid gap tokens defined on `:root` with `clamp()`.
- `--flex-gap` default container gap used by `.flex` and `.flex-inline`.

## Container Utilities

- `.flex` / `.flex-inline` set display mode with the configured gap.
- Direction utilities `.flex-row`, `.flex-row-reverse`, `.flex-col`, `.flex-col-reverse`.
- Wrapping utilities `.flex-wrap`, `.flex-nowrap`, `.flex-wrap-reverse`.
- Alignment helpers for main axis (`.justify-*`), cross axis (`.items-*`), and multi-line content (`.content-*`).

## Item Utilities

- Self-alignment via `.self-auto|start|end|center|baseline|stretch`.
- Sizing helpers `.flex-grow-*`, `.flex-shrink-*`, `.flex-basis-*`, shorthand `.flex-1|auto|initial|none`.
- Order helpers `.order-first`, `.order-last`, `.order-none`.

## Spacing Utilities

- Gap utilities `.gap-*`, `.row-gap-*`, `.col-gap-*` map to the custom properties.
- Common patterns: `.flex-center`, `.flex-between`, `.flex-around`, `.flex-stack`, `.flex-spread > *`.

## Responsive Modifiers

- Media-query driven classes for `sm`, `md`, `lg`, and `xl` breakpoints (e.g., `.md:flex-row`, `.lg:gap-md`, `.xl:items-center`).
- Large screens also increase base gaps for `.flex`/`.flex-inline` and `.flex-stack`.

## Usage

```tsx
<div className="flex gap-md justify-between">
  <div className="flex-item flex-grow-1">Left</div>
  <div className="flex-item">Right</div>
</div>

<div className="flex-col sm:flex-row gap-sm">
  <div className="flex-1">One</div>
  <div className="flex-1">Two</div>
</div>
```
